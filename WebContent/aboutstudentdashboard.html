About StudentDashBoard

Some important points you should remember..

1:Student can create its account Itself by using collage given registration number.

2:Student can not delete his account.

3:Student can not reset/recreate his password ,he/she need to contact to developer section in case of password reset issue.

4:Student can see which books are available in library and which books are issue by him/her.

5:Student can see his/her payment history, how much amount he/she paid and how much he/she will pay .

6:Student can not done any changes in the account record.

7:Student can not done any changes in the library record .

8:Any site related problem arises you can contact developer section .
