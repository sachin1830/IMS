About Admin Section

Some important points you should remember..

1:Admin can add/create accountant ids/account.

2:Admin can delete ids/account accountant.

3The deleted accounts/Ids can not recover ,so the accountant should be extra conscious when deleting the accounts/Ids.

4:Admin can add/create librarian account/Ids.

5:Admin can delete librarian account/Ids.

6:Admin can add/create teacher account/Ids.

7:Admin can delete teacher account/Ids

8:Admin can see which books are available for students and teachers in library.

9:Admin can not delete any books from library

10:Admin can not see any data from the account section

11:Admin can not reset/recreate any password for any accounts/Ids ,admin /owner of the account/id need to contact developer section.

12:site related problem arises you can contact developer section.
