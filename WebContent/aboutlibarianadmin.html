Some important points you should remember..

1:Librarian/Library admin can add the books.

2:Librarian/ Library admin can update the books.

3:Librarian/ Library admin can delete the books.

4:The deleted books can not recover ,so the librarian/ library admin should be extra conscious when deleting the book.

5:Librarian/ Library admin can see which books are available for the teachers and students.

6:Librarian/ Library admin can issue books.

7:Librarian/ Library admin can handle book return operation .

8:Librarian/ Library admin can not reset/recreate password itself he/she need to contact with Admin section or developer section(developer section consider first ).

9:Librarian/Library admin can not create its Id/account itself admin , section need to create id/account

10:Any site related problem arises you can contact developer section.
